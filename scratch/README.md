# Private Scratchpad — Per-Instance

Each instance has a private scratchpad directory at `scratch/<instance>/` (`archivist`, `advocate`, `synthesizer`, `curator`).

**Only that instance writes here. Other instances cannot read this directory.**

Think of it as your private notebook — raw thoughts, half-formed ideas, doubts, initial drafting. No one else sees this. The Curator does not read it. Jake only reads it if he explicitly opens the file.

**Lifecycle:**
1. Start your cycle here — write your raw thoughts before anything else
2. Distill what's worth sharing into your session file
3. Scratchpad files are ephemeral — you can overwrite them each cycle or append, your call

**Only the content you choose to write in your session file becomes visible to the other instances.**
